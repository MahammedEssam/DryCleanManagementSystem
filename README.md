## Dry Clean Management System (Backend)

Backend API built using ASP.NET Core with:
- DAL / BAL Architecture
- Stored Procedures (T-SQL)
- Logging System

### Tech Stack
- ASP.NET Core Web API
- SQL Server
- ADO.NET
- T-SQL

### How to Run
1. Restore database using DatabaseScript
2. Update connection string in appsettings.Development.json
3. Run the API
